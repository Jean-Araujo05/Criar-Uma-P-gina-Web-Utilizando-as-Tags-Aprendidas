Estrutura básica do HTML

(Volta)

A estrutura básica do HTML como o nome já diz é a base de todo o código, em outras palavras o esqueleto onde o conteúdo da página que será criada vai ser inserido.

  1. DOCTYPE: Declaração do tipo de documento, que informa ao navegador que o documento é um HTML5.
  2. html: A raiz do documento HTML, que envolve todo o conteúdo da página. O atributo "lang" especifica o idioma do conteúdo.
  3. head: Contém metadados sobre o documento, como o conjunto de caracteres, título da página e links para estilos e scripts.
    • meta: Fornece metadados sobre o documento, como a descrição e palavras-chave. Por exemplo:
      • description Uma breve descrição do conteúdo da página.
      • keywords: Palavras-chave relevantes para o conteúdo da página.
      • name: Um nome descritivo para o conteúdo da página.
      • content: O valor do metadado, que pode ser uma descrição, palavra-chave ou outro tipo de informação.
      • charset: O conjunto de caracteres usado no documento, como UTF-8.
      • etc.
      ,Os metadados são importantes para SEO e acessibilidade.
    • title: O título da página, que é exibido na aba do navegador e nos resultados de busca.

      • O título deve ser descritivo e conter palavras-chave relevantes para o conteúdo da página.

    O conteúdo da seção head é fundamental para o correto funcionamento e apresentação da página.

  4. body: A seção do documento que contém o conteúdo visível da página, como texto, imagens e outros elementos.

    5. A estrutura base do HTML é o alicerce fundamental de qualquer página web, agindo como o esqueleto que sustenta todo o conteúdo. Sem ela, o navegador não seria capaz de interpretar e renderizar as informações de forma coesa e organizada. É essa estrutura que define o tipo de documento, o idioma da página e onde o conteúdo visível e as informações meta (como título e links para folhas de estilo) devem ser localizados, garantindo que a página seja exibida corretamente e seja acessível para os usuários e motores de busca.

Sobre tags

(Volta)

As tags são os elementos fundamentais do HTML, usadas para estruturar e formatar o conteúdo da página. Elas são compostas por uma tag de abertura, que inicia o elemento, e uma tag de fechamento, que encerra o elemento. Entre essas tags, pode haver conteúdo ou outros elementos aninhados.

Algumas tags são auto-fechadas, como a tag img, que não requer uma tag de fechamento separada. As tags podem ter atributos adicionais que fornecem informações extras sobre o elemento, como o atributo "src" em uma imagem ou "href" em um link.

As tags são essenciais para a criação de páginas web, pois definem a estrutura e o estilo do conteúdo, permitindo que os navegadores interpretem e exibam as informações corretamente.

Sobre os atributos das tags

(Volta)

Os atributos são informações adicionais que podem ser adicionadas às tags HTML para fornecer mais contexto ou modificar o comportamento do elemento. Eles são sempre especificados na tag de abertura e geralmente vêm em pares de nome e valor.

Por exemplo, na tag img, o atributo src especifica a URL da imagem, enquanto o atributo alt fornece uma descrição alternativa da imagem para leitores de tela e quando a imagem não pode ser exibida.

Os atributos são importantes para a acessibilidade, SEO e para fornecer informações adicionais sobre os elementos da página.

Sobre o texto

(Volta)

O texto é o conteúdo principal de uma página web e é exibido dentro da seção body. Ele pode ser formatado usando várias tags HTML, como p para parágrafos, h1 a h6 para cabeçalhos e span para trechos de texto em linha.

Sobre as listas

(Volta)

As listas são usadas para agrupar itens relacionados e podem ser ordenadas ou não ordenadas. As listas não ordenadas são criadas com a tag ul, enquanto as listas ordenadas são criadas com a tag ol. Cada item da lista é definido com a tag li.

  • listas ordenadas

    • Listas ordenadas no HTML, representadas pela tag ol (de "ordered list"), são usadas para apresentar itens em uma sequência específica, onde a ordem é crucial para o significado ou a compreensão do conteúdo. Ao contrário das listas não ordenadas, que usam marcadores como pontos, as listas ordenadas utilizam por padrão números, letras ou algarismos romanos para indicar a posição de cada item. Assim como nas listas não ordenadas, cada item individual é definido pela tag li (de "list item").

  • listas não-ordenadas

    • Listas não ordenadas são elementos fundamentais no HTML, representadas pela tag ul (de "unordered list"). Elas são usadas para agrupar itens que não precisam seguir uma sequência específica, ou seja, a ordem em que os itens aparecem não altera o significado da lista. Cada item dentro de uma lista não ordenada é definido pela tag li (de "list item").

Sobre links

(Volta)

Links, ou hiperlinks, são a espinha dorsal da internet, conectando documentos e recursos na vastidão da World Wide Web. No HTML, eles são criados usando a tag , que significa "âncora". A função mais básica de um link é permitir que os usuários naveguem de uma página para outra, seja dentro do mesmo site ou para um site completamente diferente. A essência de um link está em dois componentes principais: o atributo href e o conteúdo visível do link.
